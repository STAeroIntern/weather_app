### 1. Install Python
### 2. Run the following below
#### Linux
    python -m venv venv
    source venv/bin/activate       
    pip install -r requirements.txt

#### Windows
    python -m venv venv
    venv\Scripts\activate          
    pip install -r requirements.txt

### 3. Run the code in terminal/command prompt: 
    python app.py
### 4. Open the app on http://127.0.0.1:5000
